Project Details

Project Name: IoT Weather Monitoring System

Controller: ESP32 Development Board

Sensors: DHT11 and BMP280

Programming Software: Arduino IDE

Purpose: To measure temperature, humidity and atmospheric pressure.

Project Requirements

🧠 ESP32 Board

Main controller of the project.

🌡️ DHT11 Sensor

Measures temperature and humidity.

🌬️ BMP280 Sensor

Measures atmospheric pressure.

🧩 Breadboard

Used for circuit connections.

➿ Jumper Wires

Connect sensors with ESP32.

🔌 USB Cable

Connects ESP32 to the computer.

💻 Computer

Used for Arduino programming.

🖥️ Arduino IDE

Used to write and upload code.

📡 Wi-Fi

Required for IoT connectivity.

Libraries Required

Open Arduino IDE and go to:

Sketch → Include Library → Manage Libraries

Install These Libraries

  1. DHT Sensor Library by Adafruit
  2. Adafruit Unified Sensor
  3. Adafruit BMP280 Library
  4. Adafruit BusIO
Install all the required libraries before compiling the Arduino program.

Circuit Connections

DHT11 to ESP32

DHT11 Pin ESP32 Pin Purpose
VCC 3.3V Power supply
GND GND Ground
DATA GPIO 4 Sensor data

BMP280 to ESP32

BMP280 Pin ESP32 Pin Purpose
VCC 3.3V Power supply
GND GND Ground
SDA GPIO 21 I2C data
SCL GPIO 22 I2C clock
Check the pin labels printed on your sensor module before making the circuit connections.

ESP32 Arduino Code

weather_monitoring.ino 
#include <Wire.h>
#include <DHT.h>
#include <Adafruit_BMP280.h>

#define DHTPIN 4
#define DHTTYPE DHT11

DHT dht(DHTPIN, DHTTYPE);
Adafruit_BMP280 bmp;

void setup() {
  Serial.begin(115200);

  dht.begin();

  if (!bmp.begin(0x76)) {
    Serial.println("BMP280 sensor not found!");

    while (1) {
      delay(10);
    }
  }

  Serial.println("Weather Monitoring System Started");
}

void loop() {
  float temperature = dht.readTemperature();
  float humidity = dht.readHumidity();
  float pressure = bmp.readPressure() / 100.0F;

  if (isnan(temperature) || isnan(humidity)) {
    Serial.println("Failed to read DHT11 sensor!");

    delay(2000);
    return;
  }

  Serial.println("-----------------------------");

  Serial.print("Temperature: ");
  Serial.print(temperature);
  Serial.println(" °C");

  Serial.print("Humidity: ");
  Serial.print(humidity);
  Serial.println(" %");

  Serial.print("Atmospheric Pressure: ");
  Serial.print(pressure);
  Serial.println(" hPa");

  Serial.println("-----------------------------");

  delay(5000);
}
Open the Serial Monitor and set the baud rate to 115200 to view the sensor readings.
